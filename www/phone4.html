Connecting to Device

Task 4: Text Navigation
Part 1: Choosing navigation settings
VoiceOver lets you control whether text is read to you a paragraph, a line, a word, or a character at a time. Reading by character can help with spelling. Set the text navigation rate to characters by tapping on the text below, then changing rotor gestures to the characters setting.
Part 2: Navigating text
You are now reading by character. Select the sample text, then swipe right to read by character.
Part 3: Practice
Change the text navigation rate by changing rotor gestures. Read the next character in the sample text by swiping right.
